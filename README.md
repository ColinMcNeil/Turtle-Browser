**Turtle Browser**
=====
A lightweight & highly optimized browser based on Electron.

**Current features:**

 - Search
 - Forward/Back navigation
 - Full chromium support
 - Auto-Collapse UI components
 

>  **Install & Test for Development:**   	
>  
>  `npm install`
>
>  `npm run test-win`
>
>  Or on Mac/Linux
>  `npm run test-UNIX`


----------


>  **Build for Deployment:**   	
>
>  `npm run dist`
